# vue-cli-plugin-electron-webpack

Vue CLI plugin for electron `@vue/cli` 3.0

## Installation

    $ vue create my-app
    $ cd my-app
    $ vue add electron-webpack

## Usage

run vue, electron

    $ npm run dev

package vue, run electron dev

    $ npm run start

package vue, electron

    $ npm run pack

package vue, electron and build

    $ npm run builder
