## Installation

    $ vue create my-app
    $ cd my-app
    $ vue add electron-webpack

## Usage

run dev in a terminal

    $ npm run dev

run dev in two terminal

    bash1:$ npm run serve
    bash2:$ npm run dev:main

package vue, run electron dev

    $ npm run start

package vue, electron

    $ npm run pack

package vue, electron and build

    $ npm run builder
